My Projects

PulseMind: An AI-Powered Web System for Children with ADHD

PulseMind is an AI-powered web system assisting children aged 5–10 with ADHD through behavioral assessment and personalized interventions. The system classifies ADHD subtypes using ML models (SVM, Random Forest), offers gamified cognitive tasks, focus-enhancement tools, and progress tracking to support children, caregivers, and educators.

Smart Utils Application – Automated Testing with Playwright

Developed and executed automated end-to-end test scripts for the Smart Utils application using Playwright, ensuring reliable functionality across multiple browsers and devices. Designed and implemented robust test scenarios to validate core features, enhance regression testing efficiency, and minimize manual testing effort. Incorporated assertions for UI validation, data integrity checks, and performance monitoring, resulting in improved test coverage and faster release cycles.

Food Price Tracker Mobile App

In the 2nd year, 2nd semester of the Mobile Application Development module, a team of four developed the Food Price Tracker Mobile Application using Kotlin and Firebase. Built with Android Studio, the app provides real-time price information for food items, tracks price changes, and supports secure login for store owners and customers.

Scout Membership Management System for Colombo District Scout Association

In the 2nd year, 2nd semester IT Project module, the team developed an inventory management system with Node.js, Express.js, React.js, SCSS, and MongoDB.

RR Parking Automated Car Parking System

The RR Parking Automated Car Parking System is a collaborative project by five contributors. Built with HTML, CSS, PHP, and MySQL, the web application simplifies parking management with user authentication, CRUD operations, and space reservation features.

Fitness Social Media Platform

This project involved developing a RESTful API using Spring Boot for user authentication and post management, including features for photo and video uploads, workout plans, and meal plans. The client-side web application, built with React, provided an intuitive interface for user interactions such as likes and comments.

A Role-Based Project Module Management System
